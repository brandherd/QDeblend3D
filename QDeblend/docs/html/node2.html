Aim of the Program

QDeblend is designed to provide a user-friendly Graphical User Interface (GUI) to perform a deblending of a point source with specific spectral features and an underlying spatially resolved component for 3D datacubes obtained with integral field spectrographs. So far, we have only considered the specific case of deblending Quasi-Stellar Objects (QSOs) or unobscured Active Galactic Nuclei (AGN) from their underlying host galaxies, but other applications may be possible. Please read the description of the algorithm, especially the section about advantages and limitation, in order to use the program most efficiently.
