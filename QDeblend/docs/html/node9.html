The Iterative Scheme

Our solution is an iterative scheme to reduce the host contamination of the QSO spectrum in an empirical way as good as possible. When we subtract the scaled QSO spectrum from each spaxel in the datacube during the initial iteration. The signal in the central spaxel of the residual datacube will then be by design. Since the host galaxy emission is spatially extended, their normalised scale factors will be higher than that of the QSO and for all spaxels around the centre. Some level of host galaxy emission will therefore remain in the residual datacube around the QSO position. We define a certain region around the one used to construct the QSO spectrum, i.e. 8 spaxels around the central one or any other selection of spaxels that appears best suited, to extract a host galaxy spectrum from the residual datacube. The idea is now to decontaminate the QSO spectrum iteratively by subtracting the extracted host spectrum from the QSO spectrum

Here, we have made the implicit assumption that host continuum and line emission follow the same light distribution , which is not necessarily true, but in most cases only can be constrained externally from broad-band images with an analytic profile. If no external information is available at all, the flux of the host galaxy spectrum needs to be extrapolated based on other assumption as discussed in the next paragraph.

In each of the subsequent iterations a cleaned QSO spectrum is created via and subtracted from the initial datacube. After a few iterations (3-4) the QSO spectrum usually converges to a stable solution.
