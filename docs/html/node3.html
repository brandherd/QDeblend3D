Requirements

The software is based on Python and several Add-On packages as well as Qt for the cross-platfrom User Interface framework with its PyQt bindings.
QDeblend requires the following packages to be installed on the system
  • Python 2.5.4 or later (no support for Python 3 yet)
    Download from www.python.org
  • scipy 0.7.1 and numpy 1.3.0 or later
    Download from www.scipy.org, but we strongly recommend to install those packages with the installation packages provided for your current Linux/Unix distribution.
  • pyfits 2.2.2 or later
    Download from http://www.stsci.edu/resources/software_hardware/pyfits
  • matplotlib 0.99.0 or later
    Download from http://matplotlib.sourceforge.net/
  • Qt 4.5.3 or later
    Download from http://qt.nokia.com/
  • PyQt 4.7.3 and SIP 4.11.2 or later
    Download from http://www.riverbankcomputing.co.uk/software/pyqt/intro
